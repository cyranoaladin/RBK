\section{Stack de Développement (Cyborg-Ready)}

\subsection{Environnement Local}
\begin{itemize}
    \item \textbf{OS :} Linux (Ubuntu/Pop!\_OS) ou macOS. Windows via WSL2 uniquement.
    \item \textbf{IDE :} VSCode (Profile RBK : Rust, Solidity, GitLens, Copilot).
    \item \textbf{Terminal :} Alacritty + Tmux + Starship.
\end{itemize}

\subsection{Chain Stack}
\begin{itemize}
    \item \textbf{Solana :} Rust 1.75+, Anchor 0.29+, Solana CLI 1.18+.
    \item \textbf{EVM :} Foundry (Forge, Cast, Anvil).
    \item \textbf{Indexing :} The Graph (EVM), Helius/Shyft (Solana).
\end{itemize}

\section{Infrastructure CI/CD (Github Actions)}
Tout repo étudiant doit inclure un workflow \texttt{.github/workflows/ci.yml} standardisé :
\begin{itemize}
    \item Lint : \texttt{cargo clippy} / \texttt{solhint}
    \item Test : \texttt{cargo test} / \texttt{forge test}
    \item Audit : \texttt{cargo audit}
    \item Format : \texttt{cargo fmt --check}
\end{itemize}
