\chapter{Annexes — Gabarits opérationnels \& Stratégiques}
\label{annexe:gabarits}

Cette annexe regroupe les outils de pilotage stratégique pré-configurés pour le déploiement de RBK 2.0. Ils servent de référence pour les comités de pilotage et la gestion quotidienne.

\section{Modèle de Matrice SWOT (Analyse Stratégique)}

Ce modèle est utilisé pour évaluer le positionnement de nouvelles initiatives (ex: lancement d'une nouvelle Track spécialisée).

\begin{SWOTMatrix}[Exemple : Lancement Track "Zero-Knowledge"]
\begin{itemize}[leftmargin=*]
    \item \textbf{Expertise :} Équipe pédagogique déjà formée aux bases de la crypto.
    \item \textbf{Marque :} RBK reconnu pour la difficulté technique.
\end{itemize}
&
\begin{itemize}[leftmargin=*]
    \item \textbf{Ressources :} Manque de spécialistes ZK seniors disponibles pour le mentorat.
    \item \textbf{Matériel :} Nécessité de puissance de calcul élevée pour les preuves.
\end{itemize}
\\ \hline
\rowcolor{SolanaBlue!10}\textbf{Opportunités} & \textbf{Menaces}\\ \hline
\begin{itemize}[leftmargin=*]
    \item \textbf{Marché :} Demande explosive pour la confidentialité (Privacy).
    \item \textbf{Financement :} Grants possibles de fondations (StarkWare, zkSync).
\end{itemize}
&
\begin{itemize}[leftmargin=*]
    \item \textbf{Régulation :} Incertitude sur le statut des "Privacy Coins" et outils d'anonymisation.
    \item \textbf{Complexité :} Barrière à l'entrée trop haute pour les juniors.
\end{itemize}
\\ \hline
\end{SWOTMatrix}

\section{Modèle de Priorisation MoSCoW}

Utilisé pour définir le périmètre des MVP (Minimum Viable Products) développés par le Studio.

\begin{MoSCoWTable}[Exemple : Plateforme Learning Management System (LMS) V3]
\begin{itemize}[leftmargin=*]
\item \textbf{Auth :} Login via Wallet (SBT Check).
\item \textbf{Cours :} Viewer Markdown/Vidéo.
\item \textbf{Tests :} Soumission Git automatisée.
\end{itemize} &
\begin{itemize}[leftmargin=*]
\item \textbf{Social :} Profils publics élèves.
\item \textbf{Gamification :} Leaderboard temps réel.
\item \textbf{Admin :} Dashboard analytics basique.
\end{itemize} &
\begin{itemize}[leftmargin=*]
\item \textbf{Marketplace :} Offres d'emploi intégrées.
\item \textbf{Mobile :} App native iOS/Android.
\end{itemize} &
\begin{itemize}[leftmargin=*]
\item \textbf{Metaverse :} Campus virtuel 3D.
\item \textbf{AI :} Tuteur conversationnel vocal (Reporté V4).
\end{itemize}
\\ \hline
\end{MoSCoWTable}
