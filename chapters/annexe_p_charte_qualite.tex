\chapter{ANNEXE — CHARTE DE QUALITÉ \& RÈGLES D'OR}
\label{ann:charte}

RBK 2.0 repose sur un socle de valeurs non négociables. Tout manquement à ces règles entraîne une exclusion immédiate.

\section{Les 4 Commandements de l'Ingénieur RBK}
\begin{enumerate}
    \item \textbf{No Broken Windows :} Aucun code n'est mergé sur 'main' s'il contient des warnings de linter ou des TODOs non résolus.
    \item \textbf{Don't Trust, Verify :} Chaque ligne de code générée par IA doit être auditée.
    \item \textbf{Ships or Nothing :} Un projet non déployé n'existe pas.
    \item \textbf{Leave No One Behind :} Le savoir ne vaut que s'il est partagé.
\end{enumerate}
